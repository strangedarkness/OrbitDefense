\# Orbit Defense



A 3D tower defense prototype developed in Unity.



\## Gameplay

Players defend their base by placing automated towers that attack enemies moving along a predefined path.



\## Features

\- Enemy spawning system

\- Waypoint-based enemy movement

\- Tower targeting and shooting

\- Bullet damage system

\- UI displaying enemy count and base health



\## Controls

The game runs automatically demonstrating the tower defense mechanics.
